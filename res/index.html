Demo OAuth Service

This page lists all the available components and services installed and some sample user accounts configured.

The code for the script that installs this environment can be found here.

If you want to read more on OAuth, reading the specification is recommended. It is easy to read and comprehend if you have some basic understanding of web architecture:

  • The OAuth 2.0 Authorization Framework
  • The OAuth 2.0 Authorization Framework: Bearer Token Usage
The first one deals with communication between the client and the authorization server. The second one deals with the communication between the client and the resource server.
